Overview

Day-Before-Payday Metric

Adjust future transactions by a few days to align projections with real-world banking delays such as weekends and holidays.

Import Transactions via CSV

CSV import is the fastest way to populate the app.

  • Date (yyyy-mm-dd)
  • Amount (9.99)
  • Income/Expense (expense or income)
  • Category (create these first)
  • Description (text)
  • Frequency (Monthly / 4-Weekly)
Headers and values are case-sensitive and must match exactly.

Add Transactions

You can have 4 different 'types' of transactions

When adding, the popup allows you to select either Irregular, Monthly or 4-Weekly

If you chose an end date for the transaction, this becomes a targeted transaction

Most transactions would be Monthly or 4-Weekly. A Targeted transaction is one with a Start and End date

This is mean for, eg, if you plan to have savings between two dates.


Click Edit/Add Categories & Transactions button, under Add Transaction enter relevant details

an Irregular is for one off income or expense transactions. You can havs as many Irregular transactions as needed
